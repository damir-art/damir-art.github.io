# Язык Markdown

Markdown (произносится маркдаун) - это облегчённый язык разметки.

## Заголовки
# \# Первый уровень
## \#\# Второй уровень
### \#\#\# Третий уровень
#### \#\#\#\# Четвёртый уровень

## Форматирование текста

\**курсив*\*

\*\***жирный**\*\*

\``Код внутри строки`\`

    Код после 4х пробелов

\~\~~~Зачеркнутый~~\~\~

## Списки

* элемент маркированного списка
- элемент ненумерованного списка
+ буллеты
